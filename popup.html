Memory Usage

Total JS Heap Size

The total allocated heap size, in bytes. This is the total amount of memory currently allocated for JavaScript.

Used JS Heap Size

The currently active segment of JS heap, in bytes. This value shows how much memory JavaScript code is currently using.

JS Heap Size Limit

The maximum size of the heap, in bytes, that is available to the context. This is the maximum amount of memory that JavaScript can use for its execution in the browser. This limit is set at the browser level and remains fixed during the session.
